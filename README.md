# 🎮 Steam Market Helper

[![Chrome Web Store](https://img.shields.io/badge/Chrome-Extension-green.svg)]()
[![Version](https://img.shields.io/badge/Version-1.8.0-blue.svg)]()
[![License](https://img.shields.io/badge/License-MIT-orange.svg)]()

**Steam Market Helper** — это мощное и удобное расширение для Chrome, которое помогает трейдерам отслеживать свои покупки и продажи на торговой площадке Steam. Забудьте о таблицах Excel и ручных расчётах — всё автоматически, наглядно и всегда под рукой!

---

## ✨ **Возможности**

### 🛒 **Учёт покупок**
- Добавление предметов с ценой и количеством
- Бесплатные предметы (цена 0)
- Поддержка 6 валют (₴, ₽, $, €, ₸, zł)
- Автоматический расчёт общей стоимости

### 💰 **Продажа и прибыль**
- Учёт фактически полученной суммы
- Автоматический расчёт комиссии
- Расчёт прибыли и маржи в процентах
- Поддержка массовых продаж

### 📊 **Групповые операции**
- Выбор нескольких сделок чекбоксами
- Массовая продажа с одинаковой ценой
- Массовое удаление
- Горячая клавиша `B`

### 🔍 **Расширенный поиск**
- Поиск по названию предмета
- Поиск по заметкам с подсветкой
- Фильтр по статусу (куплено/продано)
- Фильтр по дате (сегодня, неделя, месяц, год)
- Свой диапазон дат
- Фильтр по цене

### 📈 **Графики и аналитика**
- Прибыль по месяцам
- Топ-10 прибыльных сделок
- Распределение по доходности
- Динамика маржи
- Автоматическое обновление

### 📋 **История сделок**
- Полная история всех операций
- Сортировка по любому столбцу
- Заметки к каждой сделке
- Статистика: общая прибыль, средняя маржа, количество предметов

### 🎨 **Интерфейс**
- Тёмная и светлая темы
- Глитч-эффект для заголовка
- Звёздный фон
- Адаптивный дизайн
- Горячие клавиши: `N`, `H`, `S`, `B`, `Esc`

### 💾 **Данные**
- Локальное хранение истории
- Синхронизация настроек через Chrome
- Экспорт/импорт JSON (полный бэкап)
